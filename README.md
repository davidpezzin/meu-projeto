Repositório central de experimentos, scripts e automações desenvolvidos diretamente em hardware embarcado (**Raspberry Pi v6l / Raspbian OS**). Este ecossistema foi projetado para operar de forma 100% headless, utilizando uma arquitetura de sincronização segura e automatizada via SSH.

---

## 🔒 Arquitetura de Integração & Segurança

O pipeline de versionamento deste projeto elimina completamente a necessidade de autenticação por senha ou tokens manuais expostos no terminal, adotando o protocolo de criptografia assimétrica **ED25519** integrado nativamente ao ecossistema do GitHub.
